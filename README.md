# Ecommerce Page

Página de ecommerce estática desarrollada como práctica de **CSS Grid** y **Flexbox**. Incluye un layout principal con áreas nombradas, una grilla de productos y tarjetas con distribución flexible.

## Demo en vivo

**[Ver proyecto en GitHub Pages](https://nicolascabello.github.io/Ecommerce-Page/)**

## Características

- Layout general con `grid-template-areas` (header, sidebar, contenido y footer)
- Grilla de productos con CSS Grid (3 columnas)
- Sidebar con categorías usando Flexbox
- Tarjetas de producto con Flexbox (imagen, texto y botón)

## Tecnologías

- HTML5
- CSS3 (Grid + Flexbox)
